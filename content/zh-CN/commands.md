# 常用指令

以下是服务器中常用的指令列表。

## 原版指令

服务器支持以下原版指令：

`/attribute` `/clear` `/effect` `/enchant` `/fill` `/gamemode` `/give` `/ride` `/save-all` `/seed` `/setblock` `/tp` `/tellraw` `/time` `/weather`

## 传送与位置

| 指令 | 说明 |
|------|------|
| `/spawn` | 回到主城 |
| `/tp` | 传送到指定位置或玩家 |
| `/sethome` | 设置家的位置 |
| `/home` | 回到家 |
| `/back` | 返回上一位置 |

## 建筑与创作

- **Axiom** - 强大的建筑辅助模组可用
- **创世神（WorldEdit）** - 用于建筑、地形调整与高效创作

## 装饰与互动

| 指令 | 说明 |
|------|------|
| `/headdb` | 获取装饰头 |
| `/hat` | 将手中物品戴在头上 |
| `/sit` | 坐下 |
| `/lay` | 躺下 |
| `/crawl` | 爬行 |

## 音乐与娱乐

- `/music` - 打开音乐控制台（需要 PlasmoVoice 模组）

## PVP 控制

- `/pvp` - 手动开启或关闭 PVP（默认禁止 PVP）

## 小游戏

| 指令 | 说明 |
|------|------|
| `/pkstart` | 跑酷模式（难度递增） |
| `/spy` | 谁是杀手 |
