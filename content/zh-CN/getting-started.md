# 新手入门

欢迎来到 Mik 米客服务器！

本服务器为创造休闲向服务器，允许任意 Mod（包含外挂），但前提是不破坏建筑与他人创作。

## 如何加入服务器

### 中国大陆玩家
1. 加入我们的 QQ 群
2. 在群聊内向管理员申请白名单
3. 等待管理员审核并回复

### 海外玩家
1. 填写申请表单
2. 等待接收邮件通知（通常在 48 小时内）
3. 审核通过后，你会收到服务器地址

## 推荐模组

为了获得更好的游戏体验，推荐安装以下模组：

1. **PlasmoVoice** - 语音聊天与听歌
2. **Xaero 地图系列** - 服务器已专门适配
   - Xaero's World Map
   - Xaero's Minimap
   - XaeroPlus
3. **ImageFrame Client** - 地图画彩色增强
4. **Axiom** - 强大的建筑辅助模组

## 首次进入服务器

当你第一次进入服务器时：

- 你会出生在主城区域
- 使用 `/spawn` 可以随时返回主城
- 使用 `/sethome` 设置你的家
- 使用 `/home` 回到你的家

## 开始创作

找到一个你喜欢的地方，开始你的建筑创作：

1. 选择合适的建造地点
2. 确保建筑与周围环境协调
3. 使用 `/sethome [名称]` 设置家的位置
4. 开始你的创造之旅！

> **提示**：服务器支持 Axiom 和创世神（WorldEdit），可以大大提升建筑效率！
