# Useful Features

## Elevator System

The server supports iron block elevators:

- **Jump** - Teleport upward
- **Sneak** - Teleport downward

## Armor Stand Editor

Use `/asedit give` to get an armor stand editor for creating armor stands in various poses for decoration.

## Firework Launcher

Use `/fireworkgun` to get a random firework launcher to add festive atmosphere to your builds.

## Building Tips

### Planning & Design

1. **Harmonize with Environment**
   - Builds must harmonize with surroundings
   - Avoid obvious discontinuities or abrupt structures
   - Site selection must be reasonable

2. **Use Building Tools**
   - Axiom mod provides powerful building assistance features
   - WorldEdit can be used for large-scale terrain adjustments
   - Use these tools wisely to improve building efficiency

### Decoration Tips

1. **Head Decorations**
   - Use `/headdb` to get various decorative heads
   - Use `/hat` to wear items on your head

2. **Armor Stand Decorations**
   - Use armor stand editor to create various poses
   - Add lively elements to your builds

## Redstone Considerations

- Light redstone devices are allowed
- Large-scale or server-lagging redstone devices are prohibited
- Minimize high-frequency operations to avoid affecting server performance

> **Tip**: Observe other players' builds and learn from their techniques!
