# Common Commands

Here's a list of commonly used commands on the server.

## Vanilla Commands

The server supports the following vanilla commands:

`/attribute` `/clear` `/effect` `/enchant` `/fill` `/gamemode` `/give` `/ride` `/save-all` `/seed` `/setblock` `/tp` `/tellraw` `/time` `/weather`

## Teleportation & Location

| Command | Description |
|---------|-------------|
| `/spawn` | Return to spawn |
| `/tp` | Teleport to specified location or player |
| `/sethome` | Set home location |
| `/home` | Return to home |
| `/back` | Return to previous location |

## Building & Creation

- **Axiom** - Powerful building assistance mod available
- **WorldEdit** - For building, terrain adjustment, and efficient creation

## Decoration & Interaction

| Command | Description |
|---------|-------------|
| `/headdb` | Get decorative heads |
| `/hat` | Wear item in hand on head |
| `/sit` | Sit down |
| `/lay` | Lie down |
| `/crawl` | Crawl |

## Music & Entertainment

- `/music` - Open music console (requires PlasmoVoice mod)

## PVP Control

- `/pvp` - Manually enable or disable PVP (PVP disabled by default)

## Minigames

| Command | Description |
|---------|-------------|
| `/pkstart` | Parkour mode (increasing difficulty) |
| `/spy` | Who is the killer |
